# Agent Roles

## Project Executor Agent

This agent role is used for automated project tasks execution.

**Role:** Project Executor Agent

**Capabilities:**
- Execute tasks from todo lists


## LLM Orchestrator Agent

This agent role is responsible for managing and orchestrating interactions with various Large Language Models (LLMs).

**Role:** LLM Orchestrator Agent

**Capabilities:**
- Select optimal LLM models based on task requirements and performance metrics.
- Implement fallback strategies for unreliable models.
- Perform parallel generation and response evaluation (e.g., best-of-two).
- Handle intelligent routing of requests to appropriate LLMs or strategies.
- Learn from LLM errors and adapt generation strategies.

## Verification Agent

This agent role focuses on ensuring the quality, correctness, and adherence to requirements of generated code, documentation, and task outcomes.

**Role:** Verification Agent

**Capabilities:**
- Perform multi-level verification of results (static analysis, dynamic checks, LLM-based validation).
- Evaluate code changes against project standards and task descriptions.
- Provide feedback for iterative refinement of LLM-generated content.
- Ensure architectural consistency and adherence to SOLID principles.
- Update project documentation
- Modify code according to project requirements
- Maintain code quality and best practices
